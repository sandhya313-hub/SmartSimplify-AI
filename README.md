# 🧠 SmartSimplify AI

SmartSimplify AI is a simple AI-powered learning app that explains difficult topics in a way that actually feels understandable.

I built this project to make learning less intimidating and more interactive. Instead of getting long complicated answers, users can choose their learning level and receive explanations that match how they want to learn.

Whether you're a beginner trying to understand a concept for the first time or someone looking for a cleaner explanation, SmartSimplify AI adapts the response accordingly.

---

## ✨ What It Can Do

- Explain complex topics in simple language
- Different explanation styles based on difficulty level
- Chat-style learning experience
- Clean and modern UI
- Markdown formatted AI responses
- Responsive design for desktop and mobile
- Fast AI responses powered by Groq

# 🚀 Running Locally

## Clone the repository

```bash
git clone https://github.com/sandhya313-hub/SmartSimplify-AI.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

---

## Environment Variables

Create a `.env` file inside the backend folder:

```env
GROQ_API_KEY=your_api_key
```

# 💡 Why I Built This

While learning technical concepts myself, I often felt that many explanations online were either too robotic or too difficult for beginners.

So I wanted to create something that feels more natural and beginner-friendly — almost like learning from someone who simplifies things instead of overcomplicating them.

This project also helped me understand:
- full-stack development
- API integration
- frontend/backend communication
- deployment workflows
- debugging real production issues
